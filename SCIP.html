Install SCIP on CSIF machine

1. Download the SCIP Optimization Suite from SCIP website. I usually download the installation script.

2. Install
libgfortran
if neccesary. I usually grab the
libgfortran.so
from a Math machine.

3. Since we have no permisson to access
/usr/lib/
folder, we need a way for
SCIP
binary to find the library: suppose
libgfortran.so
is in
~/lib/
folder, we can add
export LD_LIBRARY_PATH=/lib:/usr/lib:~/lib
to
~/.bash_profile
.
